JavaScript 是Web编程语言。它最初被创建的目的是“使网页更生动”。如今，JavaScript 不仅可以在浏览器中执行，也可以通过Node.js在服务端执行。
JavaScript 作为被应用最广泛的浏览器语言，且与 HTML/CSS 完全集成，具有独特的地位。
